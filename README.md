# HZQ-Admin(开源版)

> 基于Spring Boot，Vue的单体应用后台管理系统
## 使用环境
- Jdk 1.8

- Mysql 5.6

- Docker (可选)

## 使用技术
- Spring boot 2.0.0

- mybatis 3.4.5 

- Spring 4.3.12

- Spring MVC 4.3.12

- Maven

- Vue

- Element ui

- 用户管理 （完成）

- 角色管理 （完成）

- 字典管理 （完成）

- 菜单管理 （完成）

- 按钮管理 （完成）

- 图标管理 （完成）
